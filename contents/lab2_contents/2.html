Способи підключення стилів

  1. Inline Styles використовується для застосування стилів безпосередньо до конкретних HTML-елементів через атрибут "style".

    2. Приклад:

            
<p style="color: red; font-size: 20px;">Червоний текст розміром 20 px.</p>
  2. Internal Styles використовується для розміщення CSS-коду всередині тегу <style> у <head> секції HTML-документа.

    3. Приклад:

            
<head>
    <style>
        p {
        color: red;
        font-size: 20px;
        }
    </style>
</head>
<body>
    <p>Червоний текст розміром 20 px.</p>
</body>
  3. External Stylesheets використовується для зберігання CSS-коду в окремому файлі .css, який підключається до HTML за допомогою елементу <link>.

    4. Приклад:

            
index.html:
<head>
    <link rel="stylesheet" href="styles.css">
</head>

styles.css:
body {
    background-color: lightgray;
}

p {
    color: red;
}
