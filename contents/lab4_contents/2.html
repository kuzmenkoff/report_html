Пояснення

Для реалізації верстки сайту для мобільної та планшетної версії використовуються медіа-запити. Медіа-запити в CSS — це інструмент для застосування стилів до веб-сторінок в залежності від характеристик пристрою, таких як розмір екрану, орієнтація та роздільна здатність. Це ключовий елемент адаптивного дизайну, який дозволяє створювати веб-сайти, що коректно відображаються на різних пристроях, від мобільних телефонів до настільних комп'ютерів.

Для створення мобільної версії сайту використовуємо медіа-запит для екранів, ширина яких менше 768 пікселів. Для планшетної версії сайту використовуємо медіа-запит для екранів шириною 768-1279 пікселів. Для більш широких екранів медіа-запити не використовуються.
